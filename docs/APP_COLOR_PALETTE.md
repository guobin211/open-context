# OpenContext 配色方案

基于应用界面截图提取的配色表。

## 主色调

| 颜色名称 | HEX       | RGB                | 用途                   |
| -------- | --------- | ------------------ | ---------------------- |
| 天空蓝   | `#87CEEB` | rgb(135, 206, 235) | 装饰图片、高亮         |
| 海洋蓝   | `#00A6D6` | rgb(0, 166, 214)   | 海水、强调色、品牌主色 |
| 深海青   | `#008B8B` | rgb(0, 139, 139)   | 深色海水区域           |

## 中性色

| 颜色名称 | HEX       | RGB                | 用途           |
| -------- | --------- | ------------------ | -------------- |
| 纯白     | `#FFFFFF` | rgb(255, 255, 255) | 主内容区背景   |
| 浅灰背景 | `#F5F5F5` | rgb(245, 245, 245) | 侧边栏背景     |
| 边框灰   | `#E0E0E0` | rgb(224, 224, 224) | 分割线、边框   |
| 次要文字 | `#757575` | rgb(117, 117, 117) | 描述文字、图标 |
| 主要文字 | `#212121` | rgb(33, 33, 33)    | 标题、正文     |

## 状态色

| 颜色名称 | HEX       | RGB                | 用途         |
| -------- | --------- | ------------------ | ------------ |
| 选中蓝   | `#E3F2FD` | rgb(227, 242, 253) | 选中项背景   |
| 成功绿   | `#4CAF50` | rgb(76, 175, 80)   | 同步成功状态 |
| 悬停灰   | `#EEEEEE` | rgb(238, 238, 238) | 悬停状态背景 |

## 图片配色（海景主题）

| 颜色名称 | HEX       | RGB                | 描述       |
| -------- | --------- | ------------------ | ---------- |
| 晴空蓝   | `#5DADE2` | rgb(93, 173, 226)  | 天空主色   |
| 云白     | `#F8F9FA` | rgb(248, 249, 250) | 云朵白色   |
| 浅海蓝   | `#48C9B0` | rgb(72, 201, 176)  | 近岸浅水   |
| 深海蓝   | `#1A5276` | rgb(26, 82, 118)   | 远处深水   |
| 青绿     | `#17A589` | rgb(23, 165, 137)  | 海水过渡色 |

## CSS 变量定义

```css
:root {
  /* 品牌色 */
  --color-primary: #00a6d6;
  --color-primary-light: #e3f2fd;

  /* 强调色 */
  --color-accent-sky: #87ceeb;
  --color-accent-ocean: #00a6d6;
  --color-accent-teal: #008b8b;

  /* 中性色 */
  --color-bg-primary: #ffffff;
  --color-bg-secondary: #f5f5f5;
  --color-border: #e0e0e0;
  --color-text-primary: #212121;
  --color-text-secondary: #757575;

  /* 状态色 */
  --color-success: #4caf50;
  --color-hover: #eeeeee;
}
```

## Tailwind 配置

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: '#00A6D6',
          light: '#E3F2FD'
        },
        accent: {
          sky: '#87CEEB',
          ocean: '#00A6D6',
          teal: '#008B8B'
        },
        neutral: {
          bg: '#F5F5F5',
          border: '#E0E0E0',
          text: '#212121',
          'text-secondary': '#757575'
        }
      }
    }
  }
};
```
